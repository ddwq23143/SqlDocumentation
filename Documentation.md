
```Sql

```

#### Целые числа авто
**smallserial** имеет размер в 2 байта и может хранить до 32 767 чисел
```Sql
id smallserial primary key
```
**serial** имеет размер в 4 байта и может хранить до 2.1 млрд чисел
```Sql
id serial primary key
```
**bigserial** имеет размер в 8 байт и может хранить до 9 квадриллионов 200 триллионов
```Sql
id bigserial primary key
```
